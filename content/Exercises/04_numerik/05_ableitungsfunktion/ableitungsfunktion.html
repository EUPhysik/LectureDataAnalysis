\[ \sf y(x) = e^{-(x-2)^2} \]

Die Ausgabe k√∂nnte wie folgt aussehen.

L√∂sungsvorschlag¬∂

Die analytische Ableitung der Funktion \(\sf y(x)\) lautet

\[ \sf y'(x) = - 2(x-2)e^{-(x-2)^2} \]
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 4, 100)

Berechnen Sie die Abweichung zwischen der analytischen und numerischen Ableitung an jedem St√ºtzpunkt und stellen Sie diese graphisch dar. Verkleinern Sie den Gitterabstand \(\sf \Delta x\), z.B. um einen Faktor 4. Was f√§llt Ihnen auf?

# Definition einer Hilfsfunktion
def ableitung_f1(nx):
    # Diskretisierung des betrachteten Intervalls
    xi = np.linspace(0, 4, nx)
    dx = xi[1] - xi[0]

    # Funktionswerte an den St√ºtzstellen xi
    yi = np.exp(-(xi-2)**2)
    
    # Erstellung eines Arrays f√ºr die numerische Ableitung
    yip = np.zeros_like(yi)

    # Berechnung der Ableitung im inneren Bereich, d.h.
    # ohne die Randwerte, mit der zentralen Differenzenformel
    yip[1:-1] = (yi[2:] - yi[:-2]) / (2*dx)

    # Berechnung der Ableitung an den Randpunkten
    # mit der Vorw√§rtsdifferenzenformel
    yip[0] = (yi[1] - yi[0]) / dx
    # mit der R√ºckw√§rtsdifferenzenformel
    yip[-1] = (yi[-1] - yi[-2]) / dx
    
    return xi, yip

# Analytische Ableitung als Funktion
def ableitung_analytisch(x):
    return -2*(x-2)*np.exp(-(x-2)**2)

# Ausgabe f√ºr den L√∂sungshinweis nx = 20
xi, yip = ableitung_f1(20)
yip_analytisch = ableitung_analytisch(xi)
    
diff = yip - yip_analytisch
plt.plot(xi, diff, marker='.')
plt.grid()
plt.xlabel('x')
plt.ylabel('numerisch - analytisch')
plt.savefig('teil3a.pdf')

# Ausgabe f√ºr den L√∂sungshinweis nx = 80
xi, yip = ableitung_f1(80)
yip_analytisch = ableitung_analytisch(xi)
    
diff = yip - yip_analytisch
plt.plot(xi, diff, marker='.')
plt.grid()
plt.xlabel('x')
plt.ylabel('numerisch - analytisch')
plt.savefig('teil3b.pdf')

for nx in [20, 40, 80]:

Aufgabenteil D¬∂

Bestimmen Sie die Formel f√ºr den Vorw√§rtsdifferenzenquotienten zweiter Ordnung zur Berechnung der ersten Ableitung. Verwenden Sie daf√ºr die Taylor-Entwicklung an zwei vorw√§rtsgerichteten Punkten, d.h. \(\sf i+1, i+2\).

Die R√ºckw√§rtsformel lautet $\( \sf y'_i = \frac{3y_{i} - 4y_{i-1} + y_{i-2}}{2\Delta x} + \mathcal{O}\left(\Delta x^2\right)\)$

L√∂sungsvorschlag¬∂

Die Taylor-Entwicklungen an den Stellen \(\sf x_i + \Delta x = x_{i+1}\) bzw. \(\sf x_i + \Delta x = x_{i+1}\) lauten bis zur Ordnung \(\sf \mathcal{O}\left(\Delta x^3\right)\):

\[\sf y(x_i + \Delta x) = y_{i+1} = y_i + y'_i\Delta x + \frac{1}{2}y''_i\Delta x^2 + \mathcal{O}\left(\Delta x^3\right) \]
\[\sf y(x_i + 2\Delta x) = y_{i+2} = y_i + y'_i(2\Delta x) + \frac{1}{2}y''_i(2\Delta x)^2 + \mathcal{O}\left(\Delta x^3\right) \]

Das Ziel ist die Bestimmung des Terms \(\sf y'\) nur mit Hilfe von den Funktionswerten, d.h. \(\sf y\). Das bedeutet, dass die Terme der zweiten Ableitungen \(\sf y''\) eliminiert werden m√ºssen. Dies kann errreicht werden indem die erste Gleichung mit vier multipliziert wird und dann die zweite Gleichung davon substrahiert wird.

\[\sf 4y_{i+1} = 4y_i + 4y'_i\Delta x + 2y''_i\Delta x^2 + \mathcal{O}\left(\Delta x^3\right) \]
\[\sf y_{i+2} = y_i + 2y'_i\Delta x + 2y''_i\Delta x^2 + \mathcal{O}\left(\Delta x^3\right) \]
\[\sf \Rightarrow 4y_{i+1} - y_{i+2} = 3y_i + 2y'_i\Delta x + \mathcal{O}\left(\Delta x^3\right) \]

Die Aufl√∂sung nach \(\sf y'_i\) f√ºhrt zu

\[ \sf y'_i = \frac{-3y_i + 4y_{i+1} - y_{i+2}}{2\Delta x} + \mathcal{O}\left(\Delta x^2\right) \]

Dies ist die Vorw√§rtsdifferenzenformel f√ºr die erste Ableitung zweiter Ordnung.

Aufgabenteil E¬∂

Verwenden Sie die Formeln zweiter Ordnung f√ºr die Berechnung der Ableitung am Rand. Wie sieht nun die Abweichung zwischen der analytischen und numerischen Ableitung aus?

Die Ausgabe k√∂nnte wie folgt aussehen.

# Definition einer Hilfsfunktion, jetzt mit der Randberechnung 
# zweiter Ordnung
def ableitung_f2(nx):
    # Diskretisierung des betrachteten Intervalls
    xi = np.linspace(0, 4, nx)
    dx = xi[1] - xi[0]

    # Funktionswerte an den St√ºtzstellen xi
    yi = np.exp(-(xi-2)**2)
    
    # Erstellung eines Arrays f√ºr die numerische Ableitung
    yip = np.zeros_like(yi)

    # Berechnung der Ableitung im inneren Bereich, d.h.
    # ohne die Randwerte, mit der zentralen Differenzenformel
    yip[1:-1] = (yi[2:] - yi[:-2]) / (2*dx)

    # Berechnung der Ableitung an den Randpunkten
    # mit der Vorw√§rtsdifferenzenformel
    yip[0] = (-3*yi[0] + 4*yi[1] - yi[2]) / (2*dx)
    # mit der R√ºckw√§rtsdifferenzenformel
    yip[-1] = (3*yi[-1] - 4*yi[-2] + yi[-3]) / (2*dx)
    
    return xi, yip

# Ausgabe f√ºr den L√∂sungshinweis nx = 80
xi, yip = ableitung_f2(80)
yip_analytisch = ableitung_analytisch(xi)
    
diff = yip - yip_analytisch
plt.plot(xi, diff, marker='.')
plt.grid()
plt.title('nx=80')
plt.xlabel('x')
plt.ylabel('numerisch - analytisch')
plt.savefig('teil5.pdf')

# Vergleichende Ausgabe
xi, yip_f1 = ableitung_f1(80)
xi, yip_f2 = ableitung_f2(80)

yip_analytisch = ableitung_analytisch(xi)
    
diff_f1 = yip_f1 - yip_analytisch
diff_f2 = yip_f2 - yip_analytisch

plt.plot(xi, diff_f1, marker='.', label='erste Ordnung')
plt.plot(xi, diff_f2, marker='.', label='zweite Ordnung')

plt.legend()
plt.grid()
plt.title('nx=80')
